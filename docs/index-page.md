# 邀请页面 (index.html)

## 概述

index.html 是一个趣味邀请网页，标题为"婷，周六出来玩嘛～"。主要功能包括：

- 飘落的爱心动画背景
- 会逃跑的"不要"按钮
- 接受后播放烟花效果
- 显示表情包图片

## 交互行为

### 飘落爱心背景

- 15 个爱心 emoji（💕💗💖🩷✨）随机分布在屏幕顶部
- 每个爱心有随机大小（1~2.5rem）、动画时长（4~10s）、延迟（0~5s）
- 爱心从屏幕外飘落并旋转，循环播放

### 逃跑按钮（"不要 😤"）

触发条件：鼠标进入按钮区域

行为：
1. 按钮获得 `escaping` class，切换为 `position: fixed` 模式
2. 按钮移动到随机位置（避开屏幕边缘）
3. 按钮文案在预设列表中切换（10 种不同文案）
4. 按钮逐渐缩小（每次缩小 3%，最小 60%）
5. 追越多次越难抓住

### 接受按钮（"好呀好呀 💕"）

触发条件：点击按钮

行为：
1. 显示结果区域（"🎉 周六老地方见 🎉"）
2. 800ms 后显示表情包（1.jpg）
3. 按钮文案变为 "(ノ´ヮ`)ノ*: ・゚✧"
4. 隐藏"不要"按钮
5. 启动烟花动画

### 烟花效果

- 每次创建 2 个烟花，每 400ms 一次，共 25 次
- 每个烟花包含 60 个粒子
- 粒子有随机颜色、速度、大小、重力效果
- 粒子逐渐透明消失

## 文件依赖

| 文件 | 说明 |

| index.html | 主页面（内嵌 CSS 和 JS） |

## 维护说明

- 页面为纯静态，无需构建
- 如需修改文案，直接编辑 HTML 中的文本内容
- 如需更换表情包，替换 1.jpg 并保持文件名一致
