# OtakuMap 番剧日历

## 概述

OtakuMap 是一个高级二次元风格的番剧每日更新查看网站，从 Bangumi API 获取数据。

## 访问地址

https://ghasikey.github.io/hello.github.io/otakumap.html

## 核心功能

- 展示每日（周一~周日）更新的番剧列表
- 点击日期标签切换星期
- 点击番剧卡片查看详情弹窗（异步加载增强数据）
- 响应式设计，移动端友好

## 技术栈

- **前端**: 纯 HTML + CSS + JavaScript
- **API**: Bangumi API
  - `GET /calendar` — 每日放送（首页）
  - `GET /v0/subjects/{id}` — 条目详情（弹窗增强）
  - `GET /v0/subjects/{id}/characters` — 角色/声优
  - `GET /v0/subjects/{id}/persons` — STAFF 信息
- **字体**: Google Fonts (Noto Sans JP / Zen Maru Gothic / Inter)
- **部署**: GitHub Pages

## UI 设计

- 深色系 + 霓虹渐变背景
- 玻璃拟态卡片
- 缓慢浮动的光斑背景
- 卡片 hover 放大 + 发光边框
- 星期标签渐变高亮
- 卡片切换动画（向上滑出 + 从下方滑入）
- 弹窗底部滑入动画

## 弹窗增强设计

### 设计目标

点击卡片 → 弹窗秒开（使用 calendar 已缓存数据）→ 异步获取详情 → 填充增强区域

### 增强区域内容

1. **评分信息** — 分数 + 排名 + 评分人数
2. **热度信息** — 在看/看过/想看人数
3. **评分分布柱状图** — 纯 CSS 实现 10 分数段可视化
4. **标签云** — 前 6 个热门标签
5. **制作信息** — 导演/音乐/制作公司（从 infobox 提取）
6. **Cast 阵容** — 主角名 + 声优名
